Este projeto é uma interface simples que mostra um carrossel de produtos.  
Ela foi feita com **HTML, CSS e JavaScript**, e possui animações suaves, botões de navegação e um visual moderno.

O que o projeto faz
- Mostra produtos em formato de carrossel  
- Permite navegar entre eles usando setas  
- Cada produto tem imagem, nome, descrição e um botão  
- Possui animações e efeitos visuais para deixar a experiência mais agradável

Como usar
1. Baixe ou clone o repositório  
2. Abra o arquivo **index.html** no navegador  
3. O carrossel vai funcionar automaticamente


 Tecnologias
- HTML  
- CSS  
- JavaScript


Objetivo
Treinar animações, transições, manipulação de DOM e criação de interfaces modernas.
